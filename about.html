关于这个博客

本科的时候用hexo搭过两个博客，但一直因为惰性没能坚持更新，正好上班也算一个新的开始，遂不是用框架而是用直接写html和css的方法做了一个新的博客。博客名字“電波圏外”意即希望在互联网上“不在信号区”的地方也能够有自己的一块自留地，能够随时记录下自己在工作生活中的所见所想。

这个博客采用简洁的设计，专注于内容本身。参考https://zhuanlan.zhihu.com/p/1943562748823856367的页面风格，通过Trae辅助搭建。

所有文章都使用 Markdown 编写，托管在 GitHub Pages 上。工程完全开源，具体教程写在blog项目内，如想参考请查看readme。
